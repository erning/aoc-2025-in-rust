# aoc-2025-in-rust

Advent of Code 2025 solutions in Rust.

## Running Solutions

```bash
# Run all days
cargo run --release --

# Run specific days
cargo run --release -- 1 2 3

# Run with example inputs
cargo run --release -- --example
```
